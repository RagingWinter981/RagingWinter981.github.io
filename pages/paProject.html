Campus Events Portal

A web application to manage and track campus events with automated hour logging.

About This Project

This project was completed during my coursework at St. Mary's University. It allows students and staff to track campus events, manage participants, and log hours automatically, reducing manual tracking by 50%.

Technologies Used

  • Frontend: EJS, HTML, CSS
  • Backend: Node.js, JavaScript
  • Database: MongoDB

Features

  • User account creation and management
  • Event tracking dashboard
  • Automated hour logging for admins and PAs

Challenges & Learnings

  • Learning MongoDB: Adapting to a non-relational mindset to handle irregular event fields.
  • Working Solo: Managed full-stack development independently, balancing time and scope.
